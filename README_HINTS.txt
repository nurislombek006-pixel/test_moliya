Подсказки стартового экрана вынесены в файл intro_hints.js

Внутри 3 группы:
- micro: 100 подсказок для микроэкономики
- marketing: 100 подсказок для маркетинга
- common: 100 общих подсказок

На экране загрузки показывается 1 случайная подсказка.
Для микроэкономики выбор идёт из micro + common.
Для маркетинга выбор идёт из marketing + common.
